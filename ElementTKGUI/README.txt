Must be run in Python 2.7

Objectives:
Show understanding of Python GUI TKinter combined with Python Image Library
Show ability to parse csv

Csv of element matrix is parsed and used to construct list on left hand side
When list is clicked, image at the bottom shows the element picked
Combo elements are shown on the right hand side in a grid, based on what the combination of the two different elements are.
Ex: Wind + Earth = Sand. As a result, sand.png is one of the images loaded dynamically
