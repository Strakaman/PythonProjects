In order to use, you must edit the files, replacing the from e-mail address, to e-mail address, and password of the from user, otherwise you will fail to authenticate
Scripts also assumes e-mail being sent is from g-mail, replace SMTP server and port at bottom if another one is to be used

Also if you use g-mail, the g-mail account must allow less secure apps (such as this script) to access the account on it's behalf. See:
https://myaccount.google.com/lesssecureapps
